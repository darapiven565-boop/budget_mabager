### Shopping List Manager

Консольна програма для керування списком покупок.

#### Можливості
- Додавання товару (назва, кількість, ціна)
- Перегляд списку
- Підрахунок загальної вартості
- Збереження у файл `text.txt`
- Завантаження зі збереженого файлу
- Обробка помилок вводу

#### Формат збереження

1. Яблука - 3 x 2.5€
2. Молоко - 1 x 1.3€

Запуск програми

Запуск здійснюється виконанням файлу:  
python filename.py
