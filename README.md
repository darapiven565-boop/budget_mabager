# budget_manager
That`s a training project for learning python, a programm for managing money  

Проєкт: “Менеджер покупок”
Створити невелику консольну програму, яка допомагає користувачу планувати покупки, рахувати суму, зберігати дані у файл і завантажувати їх при повторному запуску.

Функціональність програми:

Додавання товару
Користувач вводить назву, кількість та ціну за одиницю. Дані зберігаються у вигляді словника та додаються до списку покупок.

Перегляд списку покупок
Виводиться нумерований список усіх доданих товарів у форматі:
1 : Яблука - 3 x 2.5 гривень

Підрахунок загальної суми
Автоматичне обчислення загальної вартості всіх покупок.

Збереження списку у файл (text.txt)
Дані записуються у текстовий файл у читабельному форматі.

Завантаження списку з файлу
Програма зчитує файл, обробляє рядки, формує структуру даних і відновлює список покупок.

Обробка помилок
Реалізовано обробку некоректного вводу та помилки відсутнього файлу під час завантаження.

Меню керування програмою
У користувача є вибір дій від 1 до 6, включаючи вихід із програми.

Структура збережених даних у файлі
1. Яблука - 3 x 2.5€
2. Молоко - 1 x 1.3€

Запуск програми

Запуск здійснюється виконанням файлу:
python filename.py
